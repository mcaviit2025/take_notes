How to Use These Notes

01

Regular Study

02

Last Night Prep

Focus on: "Last Night Tip" boxes, tables, and "Key Takeaway" boxes at the end of each section. The tables and diagrams give you the maximum information per minute.

Exam Day

Always start answers with the definition (concept box). Add diagrams wherever possible — figures earn extra marks. Use real-world examples to show depth.
